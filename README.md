# Shopping List App

카테고리별 분류 기능을 갖춘 쇼핑 리스트 웹 앱입니다.

## 주요 기능

- **카테고리 분류** : 과일/채소, 정육/수산, 유제품, 간식/음료, 냉동식품, 생활용품, 기타
- **필터링** : 카테고리별 필터로 원하는 항목만 보기
- **체크리스트** : 구매 완료 항목 체크 및 진행률 표시
- **로컬 저장** : LocalStorage를 활용한 데이터 자동 저장
- **반응형 디자인** : 모바일/데스크톱 모두 지원

## 사용 방법

1. `shopping-list.html` 파일을 브라우저에서 열기
2. 카테고리를 선택하고 항목을 입력한 뒤 추가 버튼 클릭
3. 체크박스로 구매 완료 표시, X 버튼으로 삭제

## 기술 스택

- HTML / CSS / JavaScript (Vanilla)
- Google Fonts (Noto Sans KR)
